# GIFTS_SYSTEM Documentation

## Overview
The Gifts System is a comprehensive feature designed to enhance the user experience by allowing users to send and receive gifts within the platform. This document provides an overview of the key components of the Gifts System.

## 1. Sending Gifts from Profile
Users can send gifts directly from their profile. The process involves:
- Navigating to the ‘Gifts’ section on the profile page.
- Choosing a specific gift to send.
- Selecting the recipient from the friends’ list.
- Completing the transaction with an optional message.

## 2. Gift Selection Page
The Gift Selection Page displays all available gift types along with their respective prices. Key features include:
- Categorization of gifts (e.g., birthday, congratulations, etc.).
- A search bar for quick gift searches.
- Filters to sort gifts by price, popularity, or recency.

## 3. Gift Notifications for Recipients
Once a gift is sent, the recipient receives a notification. This includes:
- An in-app notification.
- An email alert (if enabled).
- Information about the sender and the specific gift received.

## 4. Announcements in Chat Rooms
Gift-related announcements can be posted in chat rooms, including:
- Alerts for gifts sent to users in the group.
- Special promotions for purchasing gifts.
- Events highlighting popular gifts during holidays or festivals.

## 5. Gifts Folder in Profile
Each user has a dedicated Gifts Folder in their profile, where:
- All received gifts are stored.
- Users can review past gifts, resend them, or express gratitude to the sender.
- Options to categorize gifts based on their type or occasion.

## 6. Show/Hide System for Managing Gifts
Users can manage which gifts appear publicly on their profiles using a show/hide system, allowing:
- Customization of the profile presentation.
- Control over privacy settings linked to gifts.
- Options to showcase favorite or recent gifts only.

## Conclusion
The Gifts System is designed to enrich user interaction and engagement within the platform by providing an intuitive, attractive, and manageable gifting experience for all users.
